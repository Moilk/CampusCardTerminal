### 系统表结构
- 学生基本信息表
	+ 学号
	+ 姓名
	+ 性别
	+ ...

- 校园卡基本信息表
	+ 卡号
	+ 学号
	+ 余额
	+ 发卡时间
	+ 卡有效期
	+ 发卡地点(设备)编号
	+ 工作人员编号
	+ ...

- 校园卡充值记录表
	+ 卡号
	+ 充值金额
	+ 充值时间
	+ 充值地点(设备编号)
	+ 工作人员编号
	+ ...

- 学生消费信息表
	+ 卡号
	+ 消费金额
	+ 消费时间
	+ 消费地点(设备)编号
	+ ...

- 地点(设备)信息表
	+ 地点(设备)编号
	+ 地点名称
	+ ...

- 系统用户表
	+ 人员编号
	+ 人员姓名
	+ 性别
	+ ...
